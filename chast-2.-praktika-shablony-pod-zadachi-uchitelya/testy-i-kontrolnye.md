# Тесты и контрольные

Создание качественных контрольных работ — одна из самых рутинных, но критически важных задач учителя. Генеративный ИИ позволяет автоматизировать этот процесс, но результат напрямую зависит от двух факторов: структуры промпта и правильного выбора модели под предметную область. Российские модели (ЯндексGPT, GigaChat) превосходят в работе с русским языком и культурным контекстом, тогда как китайские (DeepSeek, Qwen) демонстрируют лучшие результаты в логических и математических задачах.

> ⚠️ **Важно**: Никогда не используйте сгенерированные тесты без проверки фактов. Все модели склонны к «галлюцинациям» — генерации вымышленных дат, имён, цитат или математических ошибок. Проверяйте 2–3 ключевых факта в каждом тесте.

***

#### Шаблон для математики: дроби и уравнения

**Рекомендуемая модель**: **DeepSeek** или **Qwen**

Почему: эти модели обладают развитыми способностями к логическому рассуждению (reasoning), что критично для корректной генерации математических задач и решений. Российские модели (ЯндексGPT/GigaChat) чаще допускают ошибки в формулировках условий и решениях.

Ты — методист по разработке математических тестов для 5-го класса.\
Задача: Создай контрольную работу по теме «Обыкновенные дроби» из 8 заданий.\
Инструкция:

```
4 задания на выбор одного правильного ответа из 4 вариантов
2 задания на установление соответствия (дробь ↔ десятичная запись)
2 задания с развёрнутым ответом (сложение/вычитание дробей с разными знаменателями)
Для заданий с выбором ответа укажи правильный вариант и краткое решение
Для заданий с развёрнутым ответом предоставь полное решение с промежуточными шагами
```

Контекст:

```
Учебник: «Математика, 5» (авторы Н.Я. Виленкин, В.И. Жохов)
Уровень подготовки: базовый
Изученные навыки: сокращение дробей, приведение к общему знаменателю, сравнение дробей
Цель: проверить понимание основных операций с обыкновенными дробями
```

Дескрипторы:

```
Сложность: средняя (без искусственных усложнений)
Тип заданий: множественный выбор, соответствие, развёрнутый ответ
```

Форма: Таблица с колонками:\
№ | Тип задания | Текст задания | Варианты ответов | Правильный ответ | Решение

**Советы по использованию**:

* После генерации обязательно проверьте решения заданий с развёрнутым ответом — модели могут ошибаться в арифметике
* Для уравнений укажите конкретный тип (линейные, квадратные) и допустимые коэффициенты, чтобы избежать слишком сложных или тривиальных задач
* Если модель сгенерировала некорректное задание, уточните в промпте: «Избегай заданий с отрицательными числами в знаменателе»

#### Шаблон для гуманитарных предметов: вопросы с развёрнутым ответом

**Рекомендуемая модель**: **ЯндексGPT** или **GigaChat**

Почему: российские модели демонстрируют лучшее понимание стилистики русского языка, литературных произведений и исторического контекста. Однако будьте готовы к возможным отказам при работе с политически чувствительными темами (см. предупреждение ниже).<br>

Ты — старший методист по литературе с 15-летним стажем.\
Задача: Сгенерируй 12 вопросов по произведениям А.С. Пушкина для итоговой контрольной в 9 классе.\
Инструкция:

```
4 вопроса с множественным выбором (поле "type": "mcq")
4 вопроса на установление соответствия между персонажем и цитатой (поле "type": "matching")
4 вопроса с кратким ответом (поле "type": "short_answer")
Для вопросов с выбором ответа предоставь 4 варианта, где только один верный
Для вопросов на соответствие укажи 4 пары «персонаж — цитата»
Для вопросов с кратким ответом сформулируй вопросы, требующие 1–2 предложений в ответе
```

Контекст:

```
Изученные произведения: «Евгений Онегин», «Капитанская дочка», «Повести Белкина»
Учебник: «Литература. 9 класс» (авторы В.Я. Коровина, В.П. Журавлёв)
Уровень подготовки: базовый
Цель: проверить понимание сюжета, характеров персонажей и авторской позиции
```

Дескрипторы:

```
Тон: академический, но доступный для подростков
Сложность: средняя, без излишней эрудиции
Фокус: ключевые эпизоды и характеры персонажей
```

Форма: JSON-массив. Каждый объект содержит поля:

```md
"question": текст вопроса
"type": тип вопроса ("mcq", "matching", "short_answer")
"options": массив вариантов ответа (для mcq)
"correctAnswer": правильный ответ или индекс варианта
"explanation": краткое пояснение (для учителя)
```

> ⚠️ **Критическое предупреждение о цензуре**: ЯндексGPT и GigaChat отказываются отвечать на \~33% русскоязычных запросов, особенно по историческим темам (репрессии, Великая Отечественная война, современная политика). Если модель отказалась генерировать тест:
>
> 1. Переформулируйте запрос в академическом ключе: вместо «Причины репрессий 1937 года» → «Как историки объясняют социальные процессы в СССР конца 1930-х годов?»
> 2. Уберите прямые указания на политические события
> 3. В крайнем случае используйте китайские модели (Qwen/DeepSeek), но проверяйте факты — они хуже знают российскую историю

#### Практические рекомендации по работе с тестами

**Как проверять качество сгенерированного теста**

Перед использованием в классе пройдите по чек-листу:\
☐ Фактологическая точность\
☐ Проверены все даты, имена, цитаты (особенно в гуманитарных предметах)\
☐ Проверены решения математических задач (особенно в заданиях с развёрнутым ответом)\
☐ Нет вымышленных событий или персонажей («галлюцинаций»)\
☐ Педагогическая корректность\
☐ Уровень сложности соответствует возрасту учеников\
☐ Задания не содержат двусмысленных формулировок\
☐ Варианты ответов в тестах с выбором логически правдоподобны\
☐ Техническая готовность\
☐ Формат соответствует запросу (таблица, JSON, список)\
☐ Нет пропущенных колонок или полей\
☐ Текст готов к копированию в документ или систему тестирования

**Типичные ошибки моделей и как их избежать**

| Ошибка модели                  | Пример                                                                       | Как исправить в промпте                                                                                           |
| ------------------------------ | ---------------------------------------------------------------------------- | ----------------------------------------------------------------------------------------------------------------- |
| **Вымышленные цитаты**         | «Как говорил Пушкин: „Любовь — это вечный двигатель души"» (цитата выдумана) | Добавьте в контекст: «Используй только подтверждённые цитаты из произведений. Если не уверен — не приводи цитату» |
| **Математические ошибки**      | Неправильное решение уравнения в колонке «Решение»                           | Уточните: «Проверь каждое решение дважды перед выводом. Укажи все промежуточные шаги»                             |
| **Двусмысленные формулировки** | «Какой герой был самым важным?» (субъективный вопрос в тесте)                | Добавьте в инструкцию: «Избегай субъективных формулировок. Все вопросы должны иметь однозначный правильный ответ» |
| **Отказ из-за цензуры**        | Модель не генерирует вопросы по теме «Холодная война»                        | Переформулируйте: «Создай вопросы по международным отношениям 1945–1991 гг.»                                      |

**Адаптация шаблонов под другие предметы**

Шаблоны легко адаптируются под любую дисциплину. Просто замените:

1. **Роль**: «методист по литературе» → «методист по биологии»
2. **Контекст**: произведения Пушкина → тема «Клетка и её органоиды»
3. **Инструкцию**: типы вопросов оставьте те же, но измените содержание заданий
4. **Дескрипторы**: для естественных наук добавьте «Фокус: причинно-следственные связи»

**Пример быстрой адаптации для биологии**:

Ты — методист по биологии для 6 класса.\
Задача: Создай тест из 10 вопросов по теме «Строение клетки».\
Инструкция:

```
5 вопросов с выбором ответа (органоид ↔ функция)
3 вопроса на соответствие (часть клетки ↔ изображение)
2 вопроса с кратким ответом («Какую функцию выполняет ядро?»)
```

Контекст:

```
Учебник: «Биология. 5–6 классы» (автор Н.И. Сонин)
Уровень: базовый
Изучено: строение растительной и животной клетки
```

Форма: Таблица с колонками: № | Вопрос | Варианты | Ответ | Пояснение

> ✅ **Итог**: Структурированный промпт + правильный выбор модели = готовый к использованию тест за 2–3 минуты. Главное правило — всегда проверяйте факты перед применением в классе. ИИ — помощник в подготовке, но не замена педагогической экспертизе.
